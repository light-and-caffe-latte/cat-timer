# Cat Timer 🐈⏳

This is a developer's practice desktop timer application. 
The purpose of this project is to practice pushing to GitHub and to see how the project looks on the web.

## Features
- Cute design with cat icons
- Developed using Python and uv

## How to run
1. Clone the repository.
2. Run with the following command:
   ```bash
   uv run main.py
   ```

# 猫タイマー 🐈⏳

練習用のデスクトップタイマーアプリです。
GitHubへの初pushと、Web上でどのように見えるかのテストを兼ねています。

## 特徴
- 猫のアイコンを使った可愛いデザイン
- Python (uv) を使用して開発

## 使い方
1. プロジェクトをクローンします。
2. 以下のコマンドで実行します。
   ```bash
   uv run main.py
   ```
